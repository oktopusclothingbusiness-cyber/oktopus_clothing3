# **App Name**: PersonaVerse

## Core Features:

- Profile Display: Display personal profile with photo, name, and short introduction on the home page.
- Smooth Scroll: Implement smooth scrolling navigation between sections on the home page.
- Dynamic Typing Effect: Utilize AI to generate a dynamic typing effect for the hero section of the homepage, showcasing various skills (e.g., "MERN Stack Developer"). This involves the tool deciding on when and whether to show a skill.
- Timeline Display: Display a timeline-style representation of education and work experience on the 'About' page.
- Project Listing: Showcase projects with title, tech stack, GitHub link, and description, including tag-based filtering.
- Contact Form: Implement a contact form for users to submit messages and send emails via Nodemailer.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke professionalism and innovation.
- Background color: Light Gray (#F5F5F5), nearly white, for a clean and modern feel.
- Accent color: Cyan (#00BCD4) for interactive elements and highlights, providing a vibrant contrast.
- Headline font: 'Space Grotesk', sans-serif, for a techy and modern feel. Body font: 'Inter', sans-serif. 
- Use clean and minimalist icons for skills and project categories.
- Incorporate subtle animations on hover effects and page transitions to enhance user experience.
